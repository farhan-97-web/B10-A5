What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document—like an HTML document—as a tree of nodes. Each node represents a part of the document, such as elements, attributes, and text.

How do you select an element from the DOM?

Choosing the Right Method: Use getElementById() when you have a unique ID for the element. Use getElementsByTagName() when you know the tag name of the elements you want to select. Use getElementsByClassName() when you know the class name of the elements you want to select. Use querySelector() or querySelectorAll() when you want to use CSS selectors to select elements based on their attributes, relationships, or other criteria.

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation Event delegation is a technique in JavaScript where event handlers are attached to a parent element, rather than to individual child elements. This technique is particularly useful for handling events on dynamically added elements or a large number of elements. Why is it useful? By attaching a single event listener to a parent element, you reduce the number of event listeners that need to be created and managed, leading to better performance, especially for large numbers of elements.
